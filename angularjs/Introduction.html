Angularjs là 1 javascript framework, nó extend HTML attributes bằng directive và bind data to HTML bằng expression. 1. Angular directive: Angular mở rộng các attributes của HTML tag bằng directive theo cú pháp: ng-, Nếu muốn viết app hợp lệ với HTML5 ta dùng cú pháp: data-ng-. Các directive thường dùng: + ng-app: Define angular appication, chỉ định element HTML owner angular app. + ng-model: bind elements value (giống value của các input field) tới angular app. + ng-bind: bind data từ angular app tới HTML view. + ng-init: khởi tạo các variable cho angular app. 2. Angular expression: - Dùng để bind data từ application tới HTML view giống với directive ng-bind - cú pháp: {{expression}} expression này giống với javascript expression
