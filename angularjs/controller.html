Angularjs controller có tác dụng điều khiển data của angularjs ssapplication. angularjs controller là regular javascript objects. 1. Angularjs controller: - angularjs application được điều khiển bằng controllers. - ng-controller: Define application controller. - 1 controller là 1 javascript object được tạo bởi a standard javascript object constructor. - $scope: là application, html element mà nó tham chiếu đến (phạm vi của controller).
