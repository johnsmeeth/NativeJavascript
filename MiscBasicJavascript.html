1. Javascript Scope: Scope là tập hợp các biến ta có thể truy cập được hay nói cách khác scope chính là phạm vi hoạt động của các biến các hàm và các object.
2. Javascript variable: Mọi thứ trong javascript đều là object và object cũng là 1 biến nên suy ra mọi thứ trong javascript đều là biến.
3. Data types: Javascript có các primitive type như sau: String, Number, Boolean, Null, Undefined và các object type: Array, Object, Function.
4. Hoisting: Là hành vi mặc định của javascript tự động di chuyển các DECLARATIONS (không hoist INITIALIZATIONS) lên trên cùng của scope.
5. Strict mode: Là chế độ thực thi code javascript, để thiết lập strict mode đặt câu lệnh "use strict" vào đầu file .js hoặc vào đầu của scope cần thiết lập strict mode. 
6. Best Practices:
    + Thay thế việc sử dụng từ khóa new bằng các literal sau: [], {}, "", /(:)/, false, 0, function() {}.
    + Tránh sử dụng biến global.
    + Luôn khai báo biến local.
    + Sử dụng === thay cho ==.
    + Không được kết thúc definition bằng dấu comma vì dấu comma chỉ dùng để kết thúc statement.
    + Sử dụng param default.
