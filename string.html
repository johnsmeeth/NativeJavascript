String là kiểu dữ liệu dùng để lưu trữ và thao tác với text trong javascript. - Một string được tạo bởi 1 cặp single quote hoặc double quote hoặc dùng từ khóa new (object string). - Ta có thể truy cập các chracter trong 1 string thông qua index của nó - String lúc này giống như 1 mảng các character. (You can access each character in a string with its position - index) - string có thể là object: Thông thường thì string là primitive type
