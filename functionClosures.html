Các variable trong javascript có thể thuộc local scope hoặc globale scope. các biến private có thể tạo khả năng closure.
1. Global variable:
    - 1 function có thể truy cập tất cả các variable được define inside function đó.
    - 1 function cũng có thể truy cập các variable được define outside function đó.
    - variables định nghĩa trong function với từ khóa var sẽ là local variables. local variables chỉ được sử dụng bên trong function nó thuộc về.
    - varialbes được định nghĩa không bởi từ khóa var (ngay cả khi define variable inside 1 function) hoặc bên ngoài function là global variables.
    global variables thuộc về window object nên nó sẽ có thể được truy cập bởi các function khác, script code trong webpage (window) đó.

    - thực tế trong javascript function có thể truy cập tới scope above(scope bao nó).

4. javascript clusure:
    - 1 closure là 1 function có thể truy cập tới parent scope của nó ngay cả khi parent scope của nó đã close.
    - có thể hiểu nôm na khi invoke function closure như sau: vì function closure thường return 1 function vô danh khác, function vô danh này
